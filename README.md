# 🧠 Smart Resume Analyzer

An AI-powered Resume Screening Web App built using Flask and Machine Learning.

---

## 🚀 Features

- 📄 Upload Resume (PDF format)
- 🧠 AI-based Role Prediction
- 📊 ATS Score Calculation
- 🛠 Skill Extraction
- ❌ Missing Skill Detection
- 📄 Download PDF Report

---

## ⚙️ How to Run

### 1. Clone Repository
git clone https://github.com/SumanKumari0103/smart-resume-analyzer.git
cd smart-resume-analyzer

### 2. Install Dependencies
pip install -r requirements.txt

### 3. Run Project
python app.py

### 4. Open in Browser
http://127.0.0.1:5000
